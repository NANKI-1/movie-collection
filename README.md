# 个人电影收藏管理系统

## 项目介绍

电影收藏管理系统，用户可以对电影信息进行添加、查询、修改、删除操作，实现个人收藏管理。



## 技术栈

后端：Java Servlet

前端：HTML + CSS + JavaScript

数据库：MySQL

构建工具：Maven

版本控制：Git



## 开发环境

JDK 21

Tomcat 10.1.x

MySQL 5.7 / 8.0

IDE：IntelliJ IDEA



## 项目结构

movie-collection/

├── src/                      # 源代码

│   └── main/

│       ├── java/             # Java 代码

│       ├── resources/        # 配置文件

│       └── webapp/           # 页面、WEB‑INF

├── pom.xml                   # Maven 依赖

├── .gitignore                # Git 忽略文件

└── README.md                 # 项目说明



## 运行说明

1. 克隆项目到本地
2. 配置 JDK 21 + Tomcat 10.1.x
3. 创建 MySQL 数据库，修改 db.properties 连接信息
4. 使用 IDEA 打开项目，配置 Tomcat 服务器
5. 启动项目访问：http://localhost:8080/movie-collection
