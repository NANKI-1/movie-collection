每日工作流程（回顾）
====================

1. 每天开始工作前：git pull origin main 拉取最新代码
2. 基于main创建自己的功能分支：git checkout -b feature/xxx
3. 开发完成后提交：git add . → git commit -m "说明"
4. 推送远程：git push origin feature/xxx
5. 在GitHub上创建Pull Request，由另一人审核后合并到main
